lecture directe navigateur · bonus

Le navigateur ne voit ni iPhone ni Android. Mais il peut lire la batterie de cette machine (si supporte) et le niveau de certains peripheriques USB/HID (souris, casque). C'est limite, mais ca tourne sans serveur.

Servi en local par server.py. L'UI lit le JSON de /api/scan ; c'est le backend Python qui fait la lecture privilegiee (usbmux / adb / IOKit). Voir le README pour le detail appareil par appareil et la feuille de route.
